# Limpeza de Dados de Clientes Bancários

Projeto focado em limpeza e transformação de dados utilizando Python e Pandas.

## O que foi feito
- Identificação de valores nulos
- Tratamento de dados faltantes
- Conversão de tipos
- Criação de colunas derivadas (renda anual, categoria de renda)
- Salvamento de dataset limpo para análise posterior

## Tecnologias
- Python
- Pandas
- NumPy
